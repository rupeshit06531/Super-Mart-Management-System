# 🛒 Super Mart Management System

## 📌 Project Overview

Super Mart Management System is a Flask-based web application developed to manage products, customers, billing, invoices, and sales reports.

---

## 🚀 Features

- Admin Login
- Dashboard
- Product Management
- Customer Management
- Billing System
- Invoice History
- Sales Report
- Export CSV
- Print Report
- Sales Chart
- Responsive UI
- Custom 404 Page

---

## 🛠 Technologies Used

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap
- Chart.js

---

## ▶️ Installation

```bash
git clone <repository-url>

cd Super-Mart-Management-System

pip install -r requirements.txt

python app.py
```
